# User Management API

## Description
This project is a User Management API built with ASP.NET Core Web API.
It was developed using Microsoft Copilot to generate, debug, and enhance code.

## Features
- CRUD operations for users
- Input validation
- Logging middleware
- Error handling middleware
- Token-based authentication middleware
- Swagger UI support

## Authentication
Add this header to requests:
Authorization: secret-token

## Endpoints
- GET /api/users
- GET /api/users/{id}
- POST /api/users
- PUT /api/users/{id}
- DELETE /api/users/{id}

## Copilot Usage
Microsoft Copilot was used to:
- Scaffold API endpoints
- Debug validation issues
- Implement middleware
- Improve error handling
